# 📌 About the Project

Healthcare information should be accessible, reliable, and available whenever people need it. However, many users struggle to obtain quick answers to basic medical questions due to limited access to healthcare professionals, long waiting times, or lack of awareness.

**HealthAI** is an intelligent healthcare assistant developed using **IBM Granite**, designed to bridge this gap by providing AI-powered conversations related to health, wellness, symptoms, nutrition, lifestyle, and preventive care.

The system uses Natural Language Processing (NLP) and Large Language Models (LLMs) to understand user queries and generate context-aware responses in real time.

> **Disclaimer:** This application is intended for educational and informational purposes only and does not replace professional medical diagnosis or treatment.

# ❗ Problem Statement

Millions of people face challenges in accessing healthcare information because of:

- Limited medical awareness
- Long waiting times
- Expensive consultations
- Shortage of healthcare professionals
- Lack of reliable online information
- Difficulty understanding medical terminology

These challenges highlight the need for an intelligent digital healthcare assistant.

# 💡 Proposed Solution

HealthAI addresses these issues by providing:

- AI-powered health conversations
- Educational medical guidance
- Symptom awareness
- Wellness recommendations
- Nutrition advice
- Healthy lifestyle suggestions

The application delivers quick, accessible, and intelligent healthcare information while encouraging users to seek professional medical care when necessary.

# 🔄 Project Workflow

1. User opens the HealthAI web application.
2. User enters a healthcare-related question.
3. Backend validates the request.
4. Query is sent to IBM Granite.
5. AI processes the question.
6. Context-aware response is generated.
7. Response is displayed instantly.

# ⚙ Installation

## Clone Repository

```bash
git clone https://github.com/pavulurisailithan/healthai-intelligent-healthcare-assistant-using-ibm-granite.git
```

## Open Project

```bash
cd healthai-intelligent-healthcare-assistant-using-ibm-granite
```

## Install Dependencies

```bash
npm install
```

or

```bash
pip install -r requirements.txt
```

## Run Project

```bash
npm start
```

or

```bash
python app.py
```
